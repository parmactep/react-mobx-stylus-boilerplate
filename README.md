# React starter kit with mobx and stylus
## Quick start
### Get the kit
```
$ git clone https://github.com/parmactep/react-mobx-stylus-boilerplate && cd react-mobx-stylus-boilerplate
```
### Install all dependencies
```
$ npm install
```
### Development
```
$ npm run dev
```
### Build
```
$ npm run build
```
